## 저는 이렇게 일합니다

- 문제를 빠르게 읽고, 팀이 움직일 수 있는 방향으로 정리합니다.
- 사용자 인터뷰와 제품 데이터를 함께 보며 가설을 좁힙니다.
- 화면 하나보다 그 뒤의 사용 맥락, 상태, 예외, 측정 기준을 함께 봅니다.
- 아이디어에서 멈추지 않고 핸드오프, 배포, 검증까지 이어 봅니다.
- 밝고 적극적으로 소통하며 낯선 도메인도 직접 파고듭니다.
